# TinyTex
TinyTex Instructions

##### Install TinyTex
```
$ wget -qO- "https://yihui.org/tinytex/install-bin-unix.sh" | sh
```
##### Install packages
```
tlmgr install "required package"
caption
fancyhdr
colortbl
pgf
mdframed
zref
needspace
floatrow
anysize
titling
biblatex
csquotes
titlesec
setspace
circuitikz
regexpatch
xstring
siunitx
sourcesanspro
textpos
listings
enumitem
```
